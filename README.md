# Operaciones — App de Cotizaciones Forestales

PWA de gestión de cotizaciones forestales para Biodiverso.

## Uso

Abrir `index.html` en el navegador, o publicar en GitHub Pages / Netlify.

## Credenciales por defecto

- Usuario: `Admin`
- Contraseña: `1981`

## Características

- Cotizaciones: Plantación, Mantención, Riego, Servicios Mixtos
- Gestión de Insumos con categorías y filtros
- Generación de PDF profesional en escala de grises
- Logo y datos de empresa editables desde Administración
- PWA instalable en móvil y desktop
- Datos guardados en localStorage

## Deploy en GitHub Pages

1. Subir este repositorio a GitHub
2. Settings → Pages → Branch: main → /root → Save
3. Acceder en: `https://TU_USUARIO.github.io/REPO_NAME/`
